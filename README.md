## 가상환경 생성
> 작업시 가상환경을 생성해주세요.
```
python -m venv venv
```

## 가상환경 활성화
> 프로젝트 루트에서 실행해주세요.
```
source venv/bin/activate
```

## 의존성 패키지 설치
> 작업시 의존성 패키지를 설치해주세요.
```
pip install -r requirments.txt
```

## 의존성 패키지 업데이트
